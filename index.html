The 5 AI Tools

Curated by @techrefine

01 // Learning

TutorAI

Generate full learning courses on any topic in minutes.

> HOW TO USE:

Enter a topic like "Data Structures." It builds modules and quizzes instantly.

02 // Notes

AudioPen

Turns your messy rambling voice notes into clean text summaries.

> HOW TO USE:

Hit record, speak your unstructured thoughts, and it rewrites them professionally.

03 // Coding

Blackbox AI

Explains code logic better than your professor.

> HOW TO USE:

Paste a code snippet you don't understand. Ask it to "debug this" or "explain line-by-line."

04 // Writing

Caktus AI

Generates full assignments, essays, and code tailored for students.

> HOW TO USE:

Input your essay prompt. Use it to unblock your writing (always double-check facts!).

05 // Presentation

Gamma App

Creates stunning presentations, documents, and webpages instantly.

> HOW TO USE:

Type "History of AI, 8 slides." It generates text, layout, and images automatically.
